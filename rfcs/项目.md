## 概述

starter 中最核心，最重要，也最基础的对象。

## 指南级别的解释

项目包括以下最基础的信息:

- 路径 path
- 启动器程序 exe_path
- 创建时间 create_at
- 更新时间 update_at
- id
- uuid
- 图标

同时它还会指定如何使用这些数据的 executer。

## 参考级别解释

项目的定义如下:

```ts
export type StarterProjectMeta = {
  path: string;
  exePath: string;
  uuid: string;
  createAt: number;
  updateAt: number;
  icon: string;
  name: string;
  description: string;
};

export type Executer = any;

export type StarterProject = {
  meta: StarterProjectMeta;
  executer: Executer;
};
```

### 项目的存储

定义一个 project 的实体类用来抽象项目，定义如下:

```rust
pub struct Model {
    pub id: i32,
    pub uuid: Uuid,
    pub path: String,
    pub exe_path: String,
    pub create_at: NaiveDateTime,
    pub update_at: NaiveDateTime,
    pub icon: String,
    pub name: String,
    pub description: String,
    pub executer: i32,
}
```

这里唯一不同的是添加了数据自增的 id 和 executer 抽象成了一个数字更好的保存。
